# Delhi Air Quality Predictor

Real-time Air Quality Index monitoring for Delhi.

## Features

- Live AQI data from WAQI API
- Real-time updates
- Interactive Gradio interface

## Running Locally

```bash
pip install -r requirements.txt
python app.py
```

Then open `http://localhost:7860`
